# DebianDDE 日更仓库
注：该仓库的包虽然新，但并不保证可以稳定使用，请注意！  
**Daily 源目前只支持 X86 架构**
## 加源方法
需要**先添加 DebianDDE Stable 源**后才可以加入该源  
加入 Stable 源方法：  
```bash
wget http://deb.debiandde.gfdgdxi.top/sources/github.sh
bash github.sh
rm github.sh
```
加入 Daily 源方法：  
```bash
wget http://daily.deb.debiandde.gfdgdxi.top/sources/github.sh
bash github.sh
rm github.sh
```
